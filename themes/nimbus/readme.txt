A bright, modern landing-page theme for SaaS and startups, with gradient accents.

== Description ==

Nimbus is a bright, conversion-oriented WordPress theme built for SaaS products,
startups, and product launches. White cards float on a soft light canvas, deep
slate ink keeps copy crisp, and a violet-to-indigo gradient powers the pill
buttons and highlights. Roomy spacing, soft card shadows, and medium rounding
give it an energetic, modern feel — no page builders, no bloat, no tracking.

Features:

* Bright SaaS/startup aesthetic with a violet-to-indigo gradient accent
* Floating white content cards with soft shadows and medium rounding
* Sticky, blurred header with a gradient pill primary button
* Custom logo, primary navigation menu, and a widgetized sidebar
* Featured images, threaded comments, and accessible :focus-visible states
* Block editor color palette, gradient, and font sizes via theme.json
* Translation-ready (text domain: nimbus)

== Installation ==

1. In wp-admin go to Appearance > Themes > Add New > Upload Theme.
2. Choose nimbus.zip and click Install Now.
3. Click Activate.
