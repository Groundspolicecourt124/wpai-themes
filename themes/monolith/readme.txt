== Description ==

Monolith is a dark, high-contrast WordPress theme for creative studios, agencies,
and portfolios. It leans into brutalist design: oversized heavy display headings,
a monospace UI, hard 1px borders with zero rounded corners, an electric accent, and
an asymmetric two-column grid that reads like a gallery wall.

Features:

* Brutalist dark palette (#0a0a0a ground, #f5f5f5 ink, #00e5a0 electric accent)
* Oversized uppercase display headings with tight letter-spacing
* Monospace interface type for a confident, technical feel
* Two-column layout with a framed, widgetized sidebar
* Custom logo, primary navigation menu, featured images, threaded comments
* Visible :focus-visible states for keyboard accessibility
* Block editor color palette and font sizes via theme.json
* Translation-ready (text domain: monolith)

== Installation ==

1. In wp-admin go to Appearance > Themes > Add New > Upload Theme.
2. Choose monolith.zip and click Install Now.
3. Click Activate.
